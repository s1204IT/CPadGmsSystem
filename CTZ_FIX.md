# CTZ が起動不可になった場合の対処方法

必須条件
- Windows 10 以降の PC
- 文鎮化したチャレンジパッドNext
- MTK META UTILITY

1. チャレンジパッド本体上部のリセットスイッチを爪楊枝等で押す
2. **MTK META UTILITY** を起動
3. `Reboot Fastboot` 横のラジオボタンを押す
4. チャレンジパッドと接続
5. [PGPT 修正版](https://github.com/s1204IT/CPadGmsSystem/releases/download/gms/a05ba-01.00.000-gms-fix-pgpt.zip) の改造ファームウェアをダウンロードし、インストールを試みる
